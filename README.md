# Mobilização Sismob

Projeto de mobilização com integração Supabase para gerenciamento de dados sísmicos.

## 📁 Estrutura do Projeto

```
├── index.html                    # Arquivo HTML principal da aplicação
├── sismob_supabase (1).sql      # Script de banco de dados Supabase
├── README.md                     # Este arquivo
└── desktop.ini                   # Configurações do Windows
```

## 🚀 Funcionalidades

- Interface web para mobilização de dados
- Integração com banco de dados Supabase
- Gerenciamento de informações sísmicas

## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML/CSS/JavaScript
- **Backend/Database**: Supabase (PostgreSQL)

## 📝 Como Usar

1. Abra o arquivo `index.html` no navegador
2. Importe o schema do banco de dados usando `sismob_supabase (1).sql`
3. Configure as credenciais do Supabase na aplicação

## 💾 Banco de Dados

Execute o arquivo `sismob_supabase (1).sql` no seu projeto Supabase para criar as tabelas e estruturas necessárias.
